# Changelog

## [1.6.0+7] - 2026-05-19

### Added
- **Gemini AI Summaries**: Automatically fetch AI-generated summaries and chemical compositions for medicines using Google Gemini (gemini-1.5-flash).
- **Settings Integration**: Securely store and manage your Gemini API Key in the Settings screen.
- **Medicine Detail Updates**: View the AI-generated summary and composition directly on the Medicine Detail screen, complete with a clean loading state.
